# burger-restaurant-html-css
Project Description
This project is a front-end burger restaurant web application.
It was created as a design and UI practice using HTML and CSS.
The goal was to design a simple restaurant ordering experience.

The project is not connected to a backend and does not include real payment or database functionality.

## Design
The UI design was created in Figma.
